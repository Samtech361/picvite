# 📸 Picvite

**Picvite** is a lightweight, privacy-first web app that makes it easy to collect and share event photos from attendees and official photographers—all in one place, for a limited time.

Whether it’s a wedding, concert, conference, or birthday, Picvite brings everyone's moments together—quickly, securely, and effortlessly.

---

## 🚀 Features

- 🧾 **Event-Based Photo Galleries**  
  Hosts can create unique events with auto-generated QR codes for access.

- 📸 **Collaborative Uploads**  
  Guests and photographers upload their event photos directly to a shared gallery.

- 📱 **QR Code Access**  
  No logins or accounts. Attendees scan a printed QR code to access the event gallery.

- ⏳ **Temporary Events**  
  Events expire after 3 days to protect privacy and minimize storage overhead.

- ❤️ **Like & Engage**  
  Guests can like their favorite photos to highlight the best moments.

- 🛡️ **Privacy-First Design**  
  Only users with the event QR can access photos. All data is deleted after expiry.

---

## 🔧 Tech Stack (Planned)

- Backend: Node.js, Express
- Database: MongoDB (or PostgreSQL)
- Storage: AWS S3 or local (dev mode)
- Frontend: React (or another lightweight framework)
- Image Handling: Sharp / Multer
- QR Generation: `qrcode` package

---

## 💡 Why Picvite?

- No app downloads
- No public photo leaks
- No need to chase photographers for weeks
- Just scan, share, and relive the memories—before they're gone
